Monthly Expense Tracker

A simple and responsive web application that helps users track their monthly expenses, calculate remaining budget, and estimate daily spending limits based on the selected month.

Features
Select any month and automatically calculate the number of days.
Set a monthly budget.
Add multiple expense entries dynamically.
Delete expense entries when needed.
Calculate total expenditure.
Display remaining budget.
Estimate how much can be spent per day for the rest of the month.
Responsive design for desktop and mobile devices.
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
How It Works
Select a month.
Enter your total monthly budget.
Add expense descriptions and amounts.
Click Calculate.
View:
Total expenditure
Remaining money
Suggested daily spending limit
